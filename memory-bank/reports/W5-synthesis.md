## Architecture Problem: Dual Audio Backends

`AudioManager.ts` uses raw Web Audio API for SFX synthesis. The spec (§27.1)
mandates Tone.js as the backend for ALL audio. The `fix-W2-B.md` decision log
acknowledged this but chose to keep raw Web Audio — contradicting the spec.
This needs a team decision: update spec to accept dual backends, or refactor
to pure Tone.js.
